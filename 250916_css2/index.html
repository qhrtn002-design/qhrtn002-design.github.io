사용자에게 더 나은 가치를 제공하기 위해, 안정적이고 확장 가능한 시스템을 구축하는 것에 깊은 흥미를 느낍니다. 새로운 기술을 배우고 적용하며 꾸준히 성장하는 개발자가 되겠습니다.

기술 스택

포트폴리오

여행 추천 프로젝트

사용자 맞춤형 여행지를 추천하는 API 서버를 개발했습니다. Spring Boot를 기반으로 RESTful API를 설계하고, MySQL을 사용하여 여행지 데이터를 효율적으로 관리했습니다.

재능 매칭 프로젝트

재능을 가진 사용자와 재능이 필요한 사용자를 연결하는 매칭 플랫폼을 개발했습니다. Spring Security를 활용하여 정교한 매칭 알고리즘을 설계했습니다.

AWS 인프라 프로젝트

개발한 웹 애플리케이션을 AWS 클라우드 환경에 배포하고 안정적으로 운영하기 위한 인프라를 구축했습니다. RDS를 통해 데이터베이스를 관리하여 안정성을 확보했습니다.
